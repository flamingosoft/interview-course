4. Что выведет программа в каждом случае и почему?
function changeX($x){
   $x += 5;
   echo $x;
}
$x = 1;
echo $x;
changeX($x);
echo $x;

Вывод:
1
6
1
Почему: потому что переменная передаётся по значению и внутренние изменения не "прокидываются" наверх/назад

5. Что выведет программа в каждом случае и почему?
function test()
{
    static $a = 0;
    echo $a;
    $a++;
}
test();
test();
test();

0
1
2
Потому что статическая функция/переменная действует по аналогии глобальной, выход из области видимости не "сбрасывает" переменную, она продолжает жить для каждого следующего вызова функции

6. Как перевернуть массив? Есть массив array(‘h’, ‘e’, ‘l’, ‘l’, ‘o’), как из него получить array(‘o’, ‘l’, ‘l’, ‘e’, ‘h’)?

- в цикле от начала до середины свапить текущий индекс начала с аналогичным индексом конца for ...  $a[$i] <=> $a[sizeof($a) - $i]
- наверняка есть готовая функция (у меня поверхностное знакомство с пхп, а курс по пхп еще впереди)
